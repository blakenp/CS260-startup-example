    FamilyBash is a party game aimed to get friends and family together for a fun time and to get them excited about family history! You can start a game and invite your friends to the same game to play trivia games and other games as well.

    FamilyBash uses a webserver programmed in Rust to help offer a great user experience by making loading times faster, allowing users to focus more on the game rather than on the loading times! :D
